# 🧠 MNIST Handwritten Digit Classification using CNN (PyTorch)

## 📌 Project Overview

This project implements a Convolutional Neural Network (CNN) using PyTorch to classify handwritten digits from the MNIST dataset. The model learns to recognize digits (0–9) from grayscale images.

---

## 🚀 Features

* Data loading and preprocessing using `torchvision`
* CNN architecture (Conv2D + ReLU + MaxPooling)
* Fully connected layers for classification
* Training loop with loss tracking
* Model evaluation on test dataset
* Accuracy calculation

---

